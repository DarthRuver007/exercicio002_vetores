# Programa para resolver o exercício de vetores abaixo.

O exercício é:
Uma escola possui N alunos, faça um algoritmo que armazene na memória todos os nomes e a nota final destes
alunos de tal forma que nenhum nome ou nota final seja perdido. A escola comporta no máximo 100 alunos. Exiba
a lista de alunos e suas notas depois da leitura. Apresente também a média das notas final dos alunos.

Desenvolvido utilizado JDK 1.8 e NetBeans 13.
